Sobre mim

Almejo assumir uma vaga de trainee ou nível júnior em tecnologia, ansiando assim oportunidades iniciais de ingresso na área, visando aplicar conhecimentos adquiridos durante graduação e integrar a equipe da empresa, com o intuito de evoluir profissional e pessoalmente.

Formação:

● Instituto Federal de Educação, Ciências e Tecnologia da Paraíba (IFPB) - Campus Campina Grande: Graduando no Curso de Bacharelado em Engenharia de Computação (2016 – 2023)

● Instituto Federal de Educação, Ciências e Tecnologia de Pernambuco (IFPE) - Campus Pesqueira: Curso Técnico em Eletroeletrônica Integrado ao ensino médio (2011 – 2015)

Qualificações e Atividades Complementares:

● PROBEXC EXTENSÃO: Programa de Iniciação Tecnológica: Uma abordagem com ênfase nas áreas de eletrônica, programação e robótica sustentável (Campina Grande - PB, Conclusão em 2019); Descrição: Desenvolvimento de atividades de ensino de programação C e de robótica com a plataforma Arduino, utilizando uma metodologia prática.

● PROBEXC EXTENSÃO: Incentivando a Busca por Carreiras na Área Tecnológica Usando Robótica Educacional (Campina Grande - PB, conclusão em 2017); Descrição: Desenvolvimento de atividades para participação na Latin American Robotics Competition (LARC), utilizando a linguagem Python e o Kit Lego de robótica.

● PRPIPG INOVAÇÂO: Melhorando o processo de ensino-aprendizagem de Teoria dos Grafos usando um jogo digital (Campina Grande - PB, conclusão em 2018); Descrição: Desenvolvimento de um aplicativo utilizando o ambiente de desenvolvimento integrado Android Studio SDK e a linguagem de programação Java, que oferece uma interface para criação de grafos e a execução de algoritmos da área.

● LAMPIÃO: Atividades de capacitação e desenvolvimento de software do Laboratório de Aplicações Móveis, Pesquisa e InovaçÃO (Campina Grande - PB, Conclusão 2021). Descrição: Desenvolvimento de aplicativos utilizando o ambiente de desenvolvimento integrado Android Studio SDK e a linguagem de programação kotlin, com o intuito de resolver alguns problemas para a comunidade em geral.
